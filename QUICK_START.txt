# QUICK START GUIDE 🚀

## Option 1: One-Click Setup (EASIEST) ⭐

1. **Double-click `run.bat`** in this folder
2. The script will:
   - Check if Node.js is installed
   - Install all dependencies automatically
   - Start the development server
   - Open your browser at http://localhost:3000
3. Done! Your site is running

**To stop:** Press `Ctrl + C` in the terminal

---

## Option 2: Step-by-Step Manual Setup

1. First, double-click `install.bat` to install dependencies
   - This only needs to be done once
   - Wait for it to complete

2. Then double-click `run.bat` to start the dev server
   - This can be done anytime after step 1

---

## Option 3: Terminal Commands (For Advanced Users)

1. Open Command Prompt or PowerShell in this folder
2. Type: `npm install`
3. Wait for completion
4. Type: `npm run dev`
5. Browser opens automatically at http://localhost:3000

---

## Prerequisites ✅

**You need Node.js installed** (free download)

- Download: https://nodejs.org/ (choose LTS version)
- Install it
- Restart your computer (recommended)

The scripts will check this automatically and tell you if it's missing.

---

## Available Commands After Setup

```
npm run dev      → Start development server (runs the site)
npm run build    → Create production build
npm run preview  → Preview production build
```

---

## Troubleshooting

**"Node.js is not installed" error**
- Download Node.js from https://nodejs.org/
- Install it
- Restart your computer
- Run the script again

**Port 3000 is already in use**
- Close other applications using that port
- Or the script will suggest an alternative port

**npm install fails**
- Delete the `node_modules` folder and `package-lock.json` file
- Run `install.bat` again
